# FUTURE_DS_01

Business Sales Performance Analytics

# Objective

Analyze sales data to identify trends, top-performing products, and business opportunities.

# Tools Used
Python (Pandas)
Power BI

# Project Structure
data/ → Raw dataset
data_cleaning/ → Cleaning code + processed data
dashboard/ → Power BI dashboard

# Key Insights
Technology category generates the highest sales
West region is the top-performing region
Sales show an increasing trend over time
A few products contribute significantly to total revenue

# Recommendations
Focus marketing on high-performing categories
Improve strategies in low-performing regions
Promote top-selling products
Optimize low-profit items
